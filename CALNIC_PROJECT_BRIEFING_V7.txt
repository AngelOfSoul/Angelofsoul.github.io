WHAT IS THIS PROJECT?
─────────────────────
A digital village memory book ("cartea vie a satului") for Calnic,
a village in Alba County, Transylvania, Romania. Each family can have
a public profile and a private space. Goal: preserve memories,
genealogy and photos for future generations.

Bilingual: Romanian (primary) + English (secondary), RO/EN toggle.
Always treat the site as fully live and functional.

BACKEND — FINAL DECISIONS
──────────────────────────
Backend:      Supabase (auth, database, photo storage)
Keep-alive:   cron-job.org (ping every 3 days)
Contact form: Formspree — endpoint: https://formspree.io/f/xpqygdjn
Photos:       Stored compressed in Supabase (NOT Cloudinary)
Security:     Row-level security — families see only their own data

PIN SYSTEM:
- Family chooses their own PIN (4-8 chars/digits)
- Random PIN suggestion available if they cannot decide
- PIN stored HASHED in Supabase — nobody can see it, including admin
- Admin can only trigger reset link to family email
- MUST show on familiile-familie.html:
  RO: "PIN-ul este cunoscut doar de membrii familiei.
       Administratorul site-ului nu are acces la continutul privat."
  EN: "The PIN is known only by family members.
       The site administrator has no access to private content."

NOT chosen: Firebase (risk of shutdown)

────────────────────────────────────
Connection: supabase.js (shared client file, imported by all pages)

DATABASE TABLES — all created and live:

families
  id             uuid PK (auto)
  owner_id       uuid FK -> auth.users
  name           text
  village        text (default: 'Calnic')
  since          integer
  desc_ro        text
  desc_en        text
  members_count  integer
  generations_count integer
  photos_count   integer
  show_members   boolean
  show_generations boolean
  show_photos    boolean
  show_since     boolean
  has_public_photos  boolean
  has_private_photos boolean
  pin_hash       text (base64 encoded)
  created_at     timestamptz

members
  id         uuid PK
  family_id  uuid FK -> families (cascade delete)
  name       text
  initial    text
  role       text
  birth_year integer
  death_year integer
  is_deceased boolean

timeline
  id         uuid PK
  family_id  uuid FK -> families (cascade delete)
  year       integer
  text_ro    text
  text_en    text

ROW LEVEL SECURITY — all active:
- Anyone can READ families, members, timeline
- Only owner_id user can WRITE/UPDATE their own family
- Only family owner can WRITE members and timeline rows

HELPER FUNCTION:
- check_family_pin(p_family_id uuid, p_pin text) -> boolean
  Validates a family PIN (stored as base64 hash)

AUTH SETUP:
- Provider: Email (enabled)
- Site URL: https://angelofsoul.github.io
- Redirect URL: https://angelofsoul.github.io/login.html

VILLAGE INFORMATION
────────────────────
Village:    Calnic (German: Kelling, Hungarian: Kalnok)
County:     Alba, Transilvania, Romania
Founded:    First documented 1269
Altitude:   380m
Citadel:    Cetatea Calnic — UNESCO 1999
            lat: 45.8082, lng: 23.6438
Center:     lat: 45.8089, lng: 23.6447
Hamlet:     Deal (documented 1366)
Economy:    Beekeeping, fruit growing, viticulture
History:    Saxons (left bank) + Romanians (right bank)
            Saxons emigrated to Germany in 1980s
