WHAT IS THIS SITE?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Calnic Online is a digital memory book ("cartea vie a satului") for
Calnic, a village in Alba County, Transylvania, Romania.

Each family from the village can create their own private page,
add family members, a family history, a timeline of important events,
and upload photos. The site is fully bilingual: Romanian + English.

  GitHub repo:  https://github.com/AngelOfSoul/Angelofsoul.github.io
  Live site:    https://calniconline.ro  ✅ LIVE AS OF 2026-03-24
  Old temp URL: https://angelofsoul.github.io (still works, redirects)
  Upload link:  https://github.com/AngelOfSoul/Angelofsoul.github.io/upload/main

ADMIN PANEL — FULL SPECIFICATION (admin.html)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OVERVIEW:
  A private page at calniconline.ro/admin.html accessible only to the
  site owner. Completely hidden from regular visitors. Matches the dark
  gold Calnic Online design. NOT linked from any other page.
  NOT bilingual — Romanian only (owner only).

PROTECTION:
  - Custom username + password chosen by the site owner
  - Stored as hardcoded hash in the HTML (not Supabase auth)
  - Session persists via localStorage until browser is closed
  - If wrong credentials → stays on login screen, no access
  - No "forgot password" — owner sets it once in the code

SECTIONS:

  1. STATISTICI (Statistics dashboard)
     - Total families registered
     - Total members added
     - Total photos uploaded
     - Total timeline events
     - New families this month
     - Numbers pulled live from Supabase

  2. FAMILII (All families)
     - Table: family name, village, registered date, member count,
       photo count, owner email, status (active/suspended)
     - Search by name or email
     - Actions per family:
         → View public profile (opens familiile-familie.html?family=ID)
         → Suspend family (hides from public directory, keeps data)
         → Restore suspended family
         → Delete family permanently (with confirmation dialog)

  3. FOTOGRAFII RAPORTATE (Reported photos)
     - Only shows photos that have been flagged by visitors
     - Each entry shows: photo thumbnail, family name, report reason,
       date reported, reporter's note
     - Actions:
         → View full photo
         → Delete photo (removes from Storage + database)
         → Dismiss report (photo is fine, remove flag)
     - If no reports: shows "Nicio fotografie raportată" message
     - NOTE: Admin never browses ALL photos — only reported ones
       This protects family privacy while allowing moderation

  4. ANUNȚURI (Site-wide announcements)
     - Simple text editor to write an announcement in RO and EN
     - Toggle ON/OFF to show/hide on homepage
     - When active: appears as a dismissible banner on index.html
       above the hero section, in gold/dark style
     - Stored in Supabase (new "announcements" table needed)
     - Examples: village events, site maintenance, new features

  5. MESAJE (Contact form submissions)
     - Note: Formspree already emails submissions to owner
     - This section shows a log of recent submissions from Supabase
       (if we store them there) or links to Formspree dashboard
     - Alternatively: skip this if Formspree email is sufficient

  6. ACTIVITATE RECENTĂ (Recent activity log)
     - Last 20 actions on the site:
         new family registered, photo uploaded, family deleted, etc.
     - Pulled from Supabase with timestamps

  7. SECURITATE (Security)
     - List of email addresses banned from registering
     - Add/remove banned emails
     - Stored in Supabase (new "banned_emails" table)

  8. CONT SUSPENDAT management
     - List of currently suspended families
     - Quick restore button for each

PHOTO REPORT BUTTON (to be added to galerie.html and familiile-familie.html):
  - Small "Raportează" (🚩) button on every public photo
  - Clicking opens a small modal:
      "De ce raportezi această fotografie?"
      Options: Conținut ilegal / Conținut ofensator /
               Încălcarea intimității / Altceva
      Optional text field for details
      Submit button
  - Report stored in Supabase (new "photo_reports" table)
  - Admin sees it immediately in Section 3 above
  - No login required to report — anyone can flag

DATABASE CHANGES NEEDED FOR ADMIN PANEL:
  New Supabase tables:
    announcements (id, text_ro, text_en, is_active, created_at)
    photo_reports (id, photo_id, reason, note, reported_at, status)
    banned_emails (id, email, banned_at, reason)
    activity_log  (id, action_type, family_id, details, created_at)

  New Supabase column on families table:
    is_suspended boolean default false

DESIGN:
  - Same dark gold theme as all other pages
  - Uses Playfair Display + EB Garamond fonts
  - No nav bar (admin page is standalone, not linked from site)
  - Has a logout button top right
  - Sections shown as tabs or collapsible panels
  - Tables with gold accent headers
  - Action buttons: red for delete/ban, amber for suspend, green for restore
